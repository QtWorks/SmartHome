<h3><font color="blue">Options bar</h3>
<p>Options bar contains profile, workday, weekend and vacation tab.<\p>
<p>
<h4><font color="blue">Profile tab</h4>
Here you can change active profile from weekly(standard) and vacation profile.<br>
<h4><font color="blue">Workday tab</h4>
Here you can edit sprinklers and jalousie state. These states will be active during workdays if profile is weekly(standard).<br>
<h4><font color="blue">Weekend tab</h4>
Here you can edit sprinklers and jalousie state. These states will be active during weekend if profile is weekly(standard).<br>
<h4><font color="blue">Weekend tab</h4>
Here you can edit sprinklers state. These states will be active if profile is vacation (during weekdays and weekend).<br>
